About Builder Dave

Hi, I'm Builder Dave, and I'll be your guide through this project. You’ll learn the skills to automate deployment workflows, use Infrastructure as Code (IaC), and host your own website on AWS S3—all while creating a cool personal website!

Project Overview

This project shows you how to automate website deployment to AWS S3 using a CI/CD pipeline with GitHub Actions.

Steps to Complete

  • Set up AWS S3 for website hosting
  • Automate deployment using GitHub Actions
  • Use CloudFormation for IaC
  • Secure your website with CloudFront and SSL
